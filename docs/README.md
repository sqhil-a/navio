# Navio Website

This folder is a static GitHub Pages site for Navio.

To publish it:

1. Push this repo to GitHub.
2. Open `Settings -> Pages`.
3. Set source to `Deploy from a branch`.
4. Choose `main` and `/docs`.
5. Save.

Update the call-to-action in `index.html` when you have an APK release,
app store listing, or hosted Flutter web build.
